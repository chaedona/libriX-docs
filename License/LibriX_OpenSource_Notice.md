## 1. 개요

LibriX 소프트웨어 제품(이하 "본 제품")은 오픈소스 소프트웨어를 포함하고 있습니다. 본 문서는 Eclipse Public License 2.0(EPL-2.0)을 포함한 오픈소스 라이센스 요건에 따라, 사용자에게 포함된 오픈소스 컴포넌트의 사용 내역 및 라이센스 조건을 고지하기 위해 작성되었습니다.

LibriX는 오픈소스 커뮤니티의 기여를 존중하며 해당 라이센스 조건을 성실히 준수합니다.

---

## 2. LibriX 아키텍처와 오픈소스 관계

LibriX 소프트웨어는 다음과 같은 구조로 구성됩니다.

```
┌──────────────────────────────────────────────────────┐
│           LibriX 독자 개발 레이어                     │
│   (GUI 관리 콘솔, REST API, 비즈니스 로직 등)         │
│         → LibriX 상용 라이센스 적용                   │
│         → 소스코드 비공개 (독립 모듈)                 │
├──────────────────────────────────────────────────────┤
│           Open Liberty 런타임 엔진                    │
│         → Eclipse Public License 2.0 적용             │
│         → 소스코드 공개 (GitHub)                      │
└──────────────────────────────────────────────────────┘
```

LibriX의 GUI 관리 레이어는 Open Liberty와 **별도의 독립 모듈**로 설계되어 있으며, EPL-2.0의 파생 저작물(Derivative Work)에 해당하지 않습니다. 이에 따라 LibriX 독자 개발 코드에 대한 소스코드 공개 의무는 없습니다.

## 4. EPL-2.0 라이센스 핵심 조건 요약

EPL-2.0은 "약한 카피레프트(Weak Copyleft)" 라이센스로, 다음과 같은 권리와 의무를 부여합니다.

### 허용 사항 (Permissions)

| 항목 | 설명 |
|------|------|
| 상업적 사용 | 상용 제품에 포함하여 판매 가능 |
| 수정 | 소스코드 수정 허용 (수정 시 해당 코드 공개 의무) |
| 배포 | 바이너리 형태로 독자 라이센스 하에 배포 가능 |
| 특허 사용 | 기여자의 특허 침해 없이 사용 가능 (Patent Grant 포함) |
| 워런티 제공 | 배포자가 별도로 자체 워런티 제공 가능 |

### 의무 사항 (Conditions)

| 항목 | 설명 |
|------|------|
| 라이센스 동봉 | 배포 시 EPL-2.0 라이센스 전문 반드시 포함 |
| 저작권 고지 | 원본 저작권 고지 유지 필수 |
| 수정 코드 공개 | EPL 코드를 수정·배포 시 수정된 소스코드 공개 |
| Indemnification | 상용 배포자는 다른 기여자 법적 보호 의무 부담 |
| 상표 사용 제한 | 기여자의 이름·상표·로고 무단 사용 금지 |

### 비적용 사항 (Non-conditions)

- LibriX 독자 개발 코드(별도 모듈)에 대한 소스코드 공개 의무 없음
- EPL 코드에 단순 링크/호출만 하는 경우 파생 저작물로 보지 않음

## 5. 오픈소스 소스코드 취득 방법

EPL-2.0 조건에 따라, 본 제품에 포함된 오픈소스 컴포넌트의 소스코드는 다음 방법으로 취득할 수 있습니다.

### Open Liberty 원본 소스코드

```
GitHub: https://github.com/OpenLiberty/open-liberty
```

공개 저장소에서 직접 다운로드 가능합니다.

### LibriX 수정 코드 (해당 시)

LibriX가 Open Liberty를 수정한 경우, 해당 수정 소스코드는 **서면 요청일로부터 30일 이내** 무상으로 제공됩니다.

**요청 방법:**

- 이메일: support@librix.io
- 제목: "[소스코드 요청] Open Liberty 수정 코드 — LibriX v___"
- 본문: 회사명, 담당자, 요청 버전

## 7. 면책 고지

오픈소스 컴포넌트는 **"있는 그대로(AS IS)"** 제공되며, 명시적·묵시적 여하한 보증도 제공되지 않습니다. LibriX는 오픈소스 컴포넌트 자체의 결함, 보안 취약점 또는 제3자 특허 침해에 대해 별도의 책임을 부담하지 않습니다.

오픈소스 컴포넌트 관련 문제 발생 시 해당 프로젝트의 이슈 트래커를 통해 커뮤니티에 보고하시기 바랍니다.
